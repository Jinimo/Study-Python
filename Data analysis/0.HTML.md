# HTML

## 1. 규칙
1. 시작 - 끝 존재 
	- `<태그>  </태그>`
	- 태그명 : `div`,  `p`, `span`, `a`, `tr`, `td` 등
2. 태그 -> 다른 태그에 속할 수 있음
3. `<태그>  화면 표시 정보  </태그>`
4. 태그 기호 내 속성값
	  -  `<태그 속성1=값1  속성2=값2> 화면 표시 정보  </태그> `


## 2. 상위구조 
|       | 상위 태그 | |하위 태그|
|---|:---:|:---:|---:|
|한 단계  | 부모 태그 | >     |자식 태그 | 
|여러 단계 | 부모 태그 | (공백)|자손 태그 |
