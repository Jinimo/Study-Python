# 자료형 

## 1. 숫자형 (Number)

|     |     |     |
|:---:|:---:|:---:|
| 정수 (integer)|`int`| 양, 음, 0 | 
| 실수 (Floating-point)| `float` | 소수점 | 
| 8진수(Octal)  | |0o, 0O|
| 16 진수(Hexadecimal)| |0x| 

### 숫자형 연산자 
1. 사식연산  
```
int(a) + int(b) 
int(a) - int(b) 
int(a) * int(b) 
int(a) / int(b) 
int(a) ** int(b)	# 거듭제곱 
int(a) // int(b)	# 몫
int(a) % int(b)	    # 나머지  (* 짝수:%2=0)  
```
